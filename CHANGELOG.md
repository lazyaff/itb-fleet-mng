## [0.1.0] - Current Development

### Added
- Initial fleet management system with Next.js 16
- Dashboard with vehicle overview
- GPS tracking integration (Teltonika devices)
- Live vehicle tracking with Leaflet maps
- Vehicle part service scheduling
- Inspection module with dynamic criteria
- Multi-user support (admin, inspector roles)
- Prisma ORM with PostgreSQL
- NextAuth.js authentication
- Vehicle usage history tracking
- Service history management
- Usage reconciliation system

### Architecture
- Next.js 16.1.6 with React 19
- TypeScript for type safety
- Tailwind CSS v4 for styling
- Prisma 6.19.2 ORM
- PostgreSQL database
- Leaflet + react-leaflet for maps
